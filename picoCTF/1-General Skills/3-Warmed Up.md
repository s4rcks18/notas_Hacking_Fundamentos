## Warmed Up
## descripción
What is 0x3D (base 16) in decimal (base 10)?
## solución
se toman los valores y cada uno se representa de manera binaria 4 bits por termino, después se toma el valor correspondiente a cada bit. es decir 128,64... etc. y se decide su valor
3      D
128 64 32 16 8 4 2 1
0      0   1   1  1 1 0 1 = 32 + 16 + 8 + 4 + 1 = 61
picoCTF{61}
